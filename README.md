House Price Prediction
This Machine Learning project predicts house prices based on features like area, bedrooms, bathrooms, and location. Various regression algorithms were used to build a model that provides accurate price predictions.

 Tech Used
Python | Pandas | NumPy | Scikit-Learn | Matplotlib

 Algorithms
Linear Regression, Random Forest, Decision Tree

Features
Data preprocessing
Model training & testing
Performance evaluation

 Run
pip install -r requirements.txt
python main.py
